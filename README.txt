SCRAPING REDBUS DATA USING SELENIUM PROJECT

Description:
	Scraping in Redbus Project aims to understand the concept of data handling and provides knowledge in selenium which is used to scrap the data and streamlit to create a user interface which helps to fetch data from the redbus's official website and displaying it in the python's interface

STEP 1: Run the "Route_Details_file1.ipynb" file to fetch the routes of 10 states
STEP 2: Run the "Bus_details_file2.ipynb" file to fetch all bus details of all states
STEP 3: Run the "MYSQL_file3.ipynb" to clean the fetched datas and store them in mysql database.
STEP 4: Run "app.py" file which contains the interface of the project using streamlit.
